# [Toy] ShortUrl 제공 서비스

## 1. 프로젝트 소개
* URL 입력시 ShortUrl로 변환하고, ShortUrl 입력시 원본 URL로 Redirect 하는 서비스

## 2. 구현 기능
* URL 전체 조회
* URL 상세 보기
* URL 변환
* URL Redirect

## 3. 기술스택
* Java, Spring Boot, Jpa, QueryDsl, Junit
